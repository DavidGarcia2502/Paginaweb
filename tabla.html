Redes imalambricas

Wifi
tecnología que permite conectar diferentes dispositivos electrónicos a una red de internet inalámbrica utilizando ondas de radio. Se utiliza para acceder a internet de forma remota sin necesidad de cables.
Datos
La conexión móvil se refiere a la capacidad de poder acceder a internet y realizar diversas actividades en línea utilizando dispositivos móviles como smartphones y tablets.
Redes inalambricas
Una red inalámbrica es una conexión entre sistemas informáticos que se establece mediante ondas electromagnéticas, sin necesidad de cables físicos.
Bluetooth
Bluetooth es una tecnología inalámbrica que permite conectar e intercambiar datos entre dos dispositivos a corta distancia. Funciona utilizando ondas de radio para transmitir datos, lo que elimina la necesidad de hilos o cables y proporciona una conexión segura
Satelital
Una red satelital es una red de comunicación de datos facilitada por uno o más satélites de comunicación en órbita terrestre.
